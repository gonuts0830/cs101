请用 PythonShell 来计算下面这个应用题：
假设你有一台可以复制金币的机器，如果放进去 20 枚金币每天最多就可以复制出 10 枚金币。现在已经有 20 枚金币了，问一年你最多会有多少金币？

{% spoiler "提示" %}  
python 运算符     

符号 | 运算
----- | -----
'+' | 加  
'-' | 减  
'*' | 乘  
'/' | 除  
{% endspoiler %}

{% spoiler "答案" %}
>>> 365 * 10
3650
>>> 20 + 365
3670
